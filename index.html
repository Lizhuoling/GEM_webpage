Train Once, Deploy Anywhere:
Realizing Data-Efficient Dynamic Object Manipulation

Zhuoling Li1   Xiaoyang Wu1   Zhenhua Xu2   Hengshuang Zhao1  
1HKU 2THU
Paper Code Model

Abstract

     Realizing generalizable dynamic object manipulation is important for enhancing manufacturing efficiency, as it eliminates the need of specialized engineering for different scenarios. To this end, imitation learning emerges as a promising paradigm, leveraging expert demonstrations to teach a policy about manipulation skills. Although the generalization of an imitation learning policy can be improved by increasing demonstrations, demonstration collection is labor-intensive. To address this problem, this paper explores whether strong generalization in dynamic object manipulation can be achieved with only a few demonstrations. Specifically, we develop an entropy-based theoretical framework to quantify the imitation learning data requirement. Based on the insights derived from this theoretical framework, we propose a manipulation system named Generalizable Entropy Machine (GEM). The extensive experiments demonstrate that GEM can generalize across diverse environment backgrounds, robot embodiments, motion dynamics, and object geometries. Notably, GEM has been deployed in a real canteen for tableware collection. Without any in-scene demonstration, it achieves over 97% success across more than 10,000 operations.

The First Dynamic Object Manipulation Simulator

     We build the first dynamic object manipulation simulator. Based on this simulator, we generate thousands of manipulation demonstrations efficiently with only a little manual effort. Thanks to the smart design of GEM, our algorithm can naturally generalize to unseen motion dynamics of manipuated objects. Therefore, during demonstration collection, we only manipulate static objects, which greatly reduces the teleoperation challenge. This video shows a few our collected demonstration samples. Notably, the original video resolution is only 640x480. For easier viewing, we enlarged it to 1080p. This enlargement results in some noise in this video.

Manipulation Generalization in Simulator

     Th developed dynamic manipulation simulator greatly facilitates algorithm development. GEM successfully generalizes across different objects, robot embodiments, and motion dynamics successfully. For example, in this video, GEM controls a UR10e robot hand to manipulate objects in diverse geometries and the success rate is 100%, although all training demonstrations are collected based on a Franka Panda robot. The robot embodiment, manipulated objects, and moving speeds of objects are unseen in the policy training dataset.

Sim-to-Real Generalization without Tuning

     Remarkably, GEM trained using only simulated demonstrations can generalize to real-world dynamic object manipulation without real-world tuning data. This video shows that GEM can pick unseen objects in the real world, even though the environment background, robot embodiment, and motion dynamics are all unseen.

Sim-to-Real Precise Manipulation

     Thanks to the strong generalization of GEM, it can even perform very precise manipulation tasks. In this video, without using real-world tuning data, GEM inserts a block into a moving container, which only allows two millimeters of position error.

Strong Promise in Applications

     If we further tune the policy with a few real-world demonstrations, GEM can achieve a nearly 100% success rate in real scenarios, even though the novel scenarios are unseen (e.g., collect tuning data in a lab environment and deploy the policy in another environment).
